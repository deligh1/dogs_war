# はじめに
- このゲームはとあるゲームをオマージュしたものです
- 現状、ゲーム内ではほとんど説明がないので、下にある「遊び方」を参照してください
- また、文字が読みづらい、配置がずれている、キャラクターが枠からはみ出しているなど、荒削りです
- ゲームバランスは調整中ですが、遊べる難易度ではあると思います
- 追記　仕様を変更したところ、かなり難しくなりました。味方の強化や敵の弱体化を行う予定です

# 準備
- pythonとpygameが必要です
- pythonでmain.pyを実行すると遊べます

# 遊び方
- 最初にタイトル画面が開きます。「はじめる」をクリックするとゲームが開始します
- すると、選択画面が開きます。灰色の四角の中から一つを選んで次に進みます
- 四角の上側には自陣の強化、下側には敵陣の強化が書かれており、選んだ物が適用されます
- 「これにする」をクリックすると、強化・編成画面に進みます。ここでは、ゴールドを消費して強化を行うことができます
- まず、左側の緑の枠の場所でキャラクターを選択できます。選択したキャラクターは枠の色が変わります
- 真ん中の青い枠ではキャラクターを強化できます
- 選択したキャラクターは、真ん中の大きめの枠に表示されます。青い枠にあるボタンで選択したキャラクターを強化できます
- 右側の赤い枠では、その他の強化ができます
- 下にあるスロットが編成です。キャラクターを取得・強化しても編成しなければ使用できません
- 左上でキャラクターを選択し、スロットをクリックすることで編成できます
- 強化・編成が完了したら、「開戦」をクリックし、バトルを開始しましょう
- バトルは全自動で進みます。見守りましょう
- 右上のボタンで倍速・10倍速・投了を行う事ができます
- 勝利すれば報酬のゴールドを得られ、次のステージに進む事ができます
- 負けるか投了する、または全ステージクリアするとゲーム終了です
- バトル終了後は、「次へ」をクリックして次の画面に進みます
- 勝利した場合、選択画面が開き、次のステージに進みます
- 敗北か全ステージクリアした場合、クリアしたステージ数は表示され、ゲーム終了です

# 用語
- 強化倍率
    - 体力・攻撃力に掛かる倍率
- 出撃数
    - そのキャラクターが出撃される数
- 初期待機
    - 一体目が出撃されるまでの待機時間
- 再生産
    - 二体目以降の待機時間
- 城体力
    - 城の体力。自陣側の城が攻撃を受けて0になると敗北、敵陣側の城を攻撃して0にすると勝利
- スロット
    - 編成の一つの枠の事
- 出撃制限
    - 同時にステージ上に存在できる味方の最大数。これを超えて出撃する事はできない
    - 出撃できないキャラクターは出撃待機状態になり、味方キャラがやられると出撃される
    - 左側のスロットから優先的に出撃される
- 報酬
    勝利した際に得られるゴールドの事
- めっぽう強い
    - 特定の敵に対してのダメージを軽減し、攻撃を増幅する
- 停止
    - 攻撃時、一定確率で攻撃した敵の動きを一定時間止める
